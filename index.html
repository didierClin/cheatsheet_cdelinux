Les commandes sur fichiers.

ls
- liste le contenu du répertoire
ls -al
- affichage formatté incluant les fichiers cachés
cd dir
- se rendre dans le répertoire dir
cd ~
- changer vers son répertoire personnel
pwd
- indique le répertoire courant
mkdir dir
- créer un répertoire dir
rm file
- supression du fichier file
rm -r dir
- Suppression du répertoire dir et de son contenu
rm -f file
- suppression forcée de file
rm -rf dir
- force remove directory dir *
cp file1 file2
- copie file1 vers file2
cp -r dir1 dir2
- copie dir1 vers dir2 ; crée dir2 si il n'existe pas
mv file1 file2
- renomme ou deplace file1 vers file2,
- si file2 est un répertoire existant
- file1 est déplacé dans file2
ln -s file link
- crée un lien symbolique link vers file
touch file
- crée ou met à jour file
cat > file
- redirige l'entrée standard vers file
more file
- affiche le contenu de file
head file
- affiche les 10 premières lignes de file
tail file
- affiche les 10 dernières lignes de file
tail -f file
- affiche le contenu de file

Gerer les process

ps
– affiche le process actif
top
– affiche l'ensemble des process actifs
kill pid
– termine le process d'id pid
killall proc
– termine tous les process proc
bg
– liste les jobs stoppés ou s'exécutant en arrière plan,
– reprend un job en attente en arrière plan,
fg
– ramène le plus récent job en premier plan,
fg n
– ramène le job 'n' en premier plan

Permissions sur les fichiers

chmod octal fichier
- change les droit d'accès à fichier en notation octale,
- les droits sont attribués séparement à l'utilisateur user,
- au groupe, ou à tous:
  • 4 – lecture (r)
  • 2 – écriture (w)
  • 1 – exécution (x)
  • Exemples:
    • chmod 777 : lecture, écriture, exécution pour tous,
    • chmod 755 : rwx pour l'utilisateur, rx pour groupe et autres.

Recherches

grep pattern files
– recherche de la chaine pattern dans le(s) fichier(s) files
grep -r pattern dir
– recherche récursive dans le répertoire dir
command | grep pattern
– recherche de pattern dans le résultat de la commande command
locate file
– find all instances of file

Infos système

date
– affiche la date et l'heure
cal
– affiche la date calendaire,
uptime
– affiche le temps depuis le boot machine,
w
– affiche qui est en ligne,
whoami
– nom de l'utilisateur sous lequel on est loggé,
finger user
– display information about
user uname -a
– affiche les informations du noyau
cat /proc/cpuinfo
– information cpu
cat /proc/meminfo
– information mémoire
man command
– affiche la page d'aide pour command
df
– show disk usage
du
– affiche l'utiilisation disque,
free
– affiche l'utilitation mémoire et swap,
whereis app
– différents chemins vers une application,
which app
– chemin de l'application lancée par défaut

Archivage

tar cf file.tar files
- crée une archive tar nommée file.tar contenant les fichiers files,
tar xf file.tar
- extrait les fichiers depuis file.tar,
tar czf file.tar.gz files
- crée une archive tar puis compresse avec Gzip,
tar xzf file.tar.gz
- extrait un fichier de l'archive compressée par Gzip,
tar cjf file.tar.bz2
- crée une archive tar puis compresse avec Bzip2,
tar xjf file.tar.bz2
- extrait un fichier de l'archive compressée par Bzip2,
gzip file
- compresse file vers file.gz
gzip -d file.gz
- décompresse file.gz vers file

Raccourcis

Ctrl+C
- Stoppe la commande en cours
Ctrl+Z
- Stoppe la commande en cours, retourne à fg ou bg
Ctrl+D
- termine la session courante ,équivalent à exit
Ctrl+W
- efface un mot dans la ligne courante
Ctrl+U
- efface la ligne
Ctrl+R
- type to bring up a recent command
!!
- répéter la dernière commande
exit
- log out de la session courante
